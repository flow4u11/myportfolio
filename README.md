# WELCOME TO MY PORTFOLIO (NOT COMPLETED)

A personal portfolio website showcasing UI/UX design work, featuring a high-level of interactivity and visual polish.

## Features

- **Dynamic Theming**: Includes a "Persona 5" inspired theme, along with Default, Light, and Christmas themes.
- **Smooth Animations**: Powered by GSAP and ScrollTrigger for seamless section transitions and entrance effects.
- **Smooth Scrolling**: Implemented using Lenis for a fluid user experience.
- **Interactive Elements**:
  - Custom target cursor with hover states.
  - Masonry grid layout for the "Works" section.
  - Interactive particle background.
  - Proximity-based text effects.
- **Responsive Design**: Fully responsive layout adjusting for desktop and mobile viewing.

## Technologies Used

- **HTML5** & **CSS3**
- **JavaScript (ES6+)**
- **Libraries**:
  - [GSAP](https://greensock.com/gsap/) (Animations)
  - [Lenis](https://github.com/studio-freight/lenis) (Smooth Scroll)

## Usage

This is a static website. You can view it by simply opening `index.html` in your web browser.

No build step is required.

## Structure

- `assets/` - General assets (fonts, icons, etc.)
- `css/` - Stylesheets
- `images/` - Portfolio images
- `js/` - JavaScript logic
- `index.html` - Main entry point
